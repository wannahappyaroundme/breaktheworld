# 반복 오류 기록

### 2026-07-16 로컬 Edge Function 준비 상태 오판

Symptom: 인증 통합 검증에서 프로필 생성과 로그인이 간헐적으로 502 응답을 받았다.

Cause: 새 `supabase functions serve` 프로세스가 준비되기 전에 이전 게이트웨이 응답을 준비 완료로 판단했다.

Fix: 자식 프로세스의 `Serving functions on` 신호와 새 함수의 정확한 400 응답을 모두 확인한 뒤 검증을 시작한다.

Prevention: 인증 통합 검증 스크립트가 런타임 신호와 응답 본문을 함께 확인하고, 끝날 때 전체 프로세스 그룹을 종료한다.
